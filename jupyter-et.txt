----------
「ETロボコンに向けたJupyter Notebookのインストール＋何か」
----------
[hogehoge]は任意の文字列(環境により異なる)。
[N]は任意の定数(環境により異なる)。
すべての[hogehoge]や[N]の中身はかならずしも同じとは限らない。
[USERNAME]は自分のユーザー名。
# の後の文字列はターミナルの出力。
$ の後の文字列は自分がターミナルに向かって入力する文字列。
! の後の文字列は注意すべきことやメモ。
----------
インストールするもの
Python 3.[N] ぱいそん
Jupyter Notebookじゅぴたーのーとぶっく
numpy なむぽよ
matplotlib まっとぷろっとりぶ
pandas ぱんだす
----------
ウェブブラウザでhttps://www.python.org/downloads/にアクセス。
サイト内の"Download Python 3.[N].[N]"をクリック。
(ダウンロード操作は、使用するブラウザにより異なるので省略)
ダウンロードされたpython-3.[N].[N].exeを起動すると、自動的にpythonのインストールが始まる。
//インストール中にユーザの操作が要求されるのは、インストール終了時にインストーラを閉じるときの一回のみ。
Add Python to PATHにチェックを入れて、Install Nowをクリック。
! 以後、Pythonやターミナルは管理者権限で実行するものとする。
ターミナル(PowerShellとか)でpyと打って実行することでPythonを起動できる。
$ py
# Python 3.[N].[N] ([hogehoge],[hogehoge])[[hogehoge]] on [hogehoge]
# Type "help", "copyright", "credits" or "license" for more information.
# >>>
>>>の後にコマンドを入力する。
quit()でPythonを終了できる。
$ quit()
# [hogehoge] >
JupyterなどのPythonを使用するツールをインストールするには、pipというパッケージマネージャを使用する。
Jupyterにスタンドアロン・インストーラは存在しない。(Windowsユーザが戸惑うやつ)
pipはPythonをインストールすると勝手についてくる。
Jupyterをインストールする前に、pipをアップグレードする。(更新が来てるかもしれないので)
pythonを終了した状態(ターミナルの画面)で、下のコマンドを入力する。
$ pip install --upgrade setuptools
$ pip install --upgrade pip
! もしpipコマンドが無いと言われたら、下のディレクトリにパスを通す。(通し方は環境によって違うので省略)
! C:\Users\[USERNAME]\AppData\Local\Programs\Python\Python[hogehoge]\Scripts
pipのアップグレードが終わったら、jupyterをインストールする。
$ pip install jupyter
Jupyterのインストール中に操作は一切不要。勝手に終了してくれる。
Jupyterのインストールが終わったら、早速起動してみる。
$ jupyter notebook
# [hogehoge]
# To access the notebook, open this file in a browser:
#    file:///C:/Users/[USERNAME]/AppData/Roaming/jupyter/runtime/nbserver-[hogehoge].html
# Or copy and paste one of these URLs:
#    http://localhost:8888/?token=[hogehoge]
# or http://127.0.0.1:8888/?token=[hogehoge]
Webブラウザで上のURLにアクセスする。(EdgeかChromeが望ましい)
Jupyterの画面が出てれば成功。
(ファイルを)Newをクリック→Python 3をクリックで新規ノートブックを作成。起動直後はC:\Users\[USERNAME]
Hello Worldしてみる。
下の２行はJupyter Notebookの画面。
$ print("Hello World")
# Hello World
正常に動作することが確認できたらJupyter Notebookを落とす。
Fileをクリック→Close and Haltをクリックで編集を終了できる。(自動保存？)
Quitをクリックでサーバーを落とす。
ターミナルに戻る。
numpyとmatplotlibとpandasをインストールする。
$ pip install numpy
$ pip install matplotlib
$ pip install pandas
環境構築終了。
