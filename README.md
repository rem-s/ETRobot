# ETRobot

### Masterプログラムの概要
##### bluetooth_w関数
Bluetooth通信待ち関数    

##### linetrace関数    
LineTrace用のturn変数の値を変更する関数   
PIDなどはこの関数に追加　　　
　　　
##### filtering関数   
センサ値フィルタリング関数　　　
ローパスフィルタなどはこの関数に追加   

##### normalization関数    
センサ値正規化関数    
正規化などはこの関数に追加    

##### position関数
自己位置の計算用の関数　　　
自己位置すいてはこの関数に追加
